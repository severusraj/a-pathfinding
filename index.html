How A* Algorithm Works

The A* algorithm finds the shortest path between a start and end point. It uses a heuristic to prioritize cells based on their estimated distance to the end. It considers both the distance already traveled (g) and the estimated distance to the goal (h), where f = g + h.

Instructions:

  • Use the buttons to set the start and end points, or use the "S" and "E" keys.
  • Left-click on cells to create obstacles; right-click to remove them.
  • Press "Space" or click "Run Algorithm" to begin pathfinding.
  • Use the slider to adjust animation speed.
  • Press "R" or click "Reset" to clear the grid.
